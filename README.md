# Music Mood/Genre Analysis

Machine Learning project focused on predicting music genres using audio features and data analysis.

## Project Goal

The objective of this project is to classify songs into different music genres using machine learning models and audio characteristics.

Genres included:
- Hip-Hop
- Pop
- Folk
- Rock
- Electronic
- Instrumental
- Experimental
- International

---

## Audio Features Used

The models were trained using audio features such as:

- MFCC
- Energy
- Danceability
- Loudness
- Spectral features
- Instrumentalness
- Valence

---

## Machine Learning Models

Several models were tested:

- Logistic Regression
- Random Forest
- SVM (RBF)
- XGBoost

Best model:
**XGBoost**

---

## Main Results

- Music genres show distinct audio patterns.
- Some genres overlap due to similar musical characteristics.
- XGBoost achieved the best performance with around 59% accuracy/F1-score.
